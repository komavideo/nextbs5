Next.js 工程模版
=================

## 安装库

```bash
$ npx create-next-app@latest nextbs5
$ cd nextbs5
# React + Bootstrap 5
$ npm install react-bootstrap bootstrap@5.1.3
# Next.js 中间件
$ npm install next-connect
```

## Github

```bash
git remote add origin https://github.com/komavideo/nextbs5.git
git branch -M main
git push -u origin main
```
